# CarWashIT

CarWashIT is a system for private usage intended to help owners of car washes in business.

<b>Database relationships</b><br>
One-to-One:
  Car - CarNumber<br>
One-to-Many:
  Client - Car<br>
Many-to-Many:
  Order-Car

Relational Database = SQLite
